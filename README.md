## 🖥️ Instalación y uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/efe13dev/portfolio-react.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el proyecto en modo desarrollo:
   ```bash
   npm run dev
   ```
4. Accede a `http://localhost:5173` para ver el portafolio.
